
### How to run the logic to reverse the file content?
1 - Check this code out.</br>
2 - Make sure MAVEN_HOME, JAVA_HOME and Path is set correct.</br>
3 - In the project folder run mvn clean install.</br>
4 - Open command prompt at the location where this code is checked out.</br>
mvn exec:java -Dexec.mainClass="org.example.App"
